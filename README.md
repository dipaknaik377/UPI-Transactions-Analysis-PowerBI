# 📊 UPI Transactions Analysis Dashboard (2024)

## 📌 Project Overview
This project involves a comprehensive analysis of **UPI (Unified Payments Interface) transactions** for the year 2024. Using **Power BI**, I have visualized financial data to uncover spending patterns, regional trends, and transaction behaviors.

## 🚀 Key Features & Insights
- **Monthly Trend Analysis:** Visualized transaction volumes and balance fluctuations using Line and Column charts.
- **Regional Spending:** Analyzed data across major cities like **Mumbai, Delhi, Bangalore, and Hyderabad**.
- **User Demographics:** Segmented data by Age Groups and Gender to understand user behavior.
- **Dynamic Filtering:** Integrated advanced slicers for **Payment Method, Purpose, and Transaction Type** for a customized view.

## 🛠️ Tech Stack Used
- **Power BI Desktop:** For data visualization and interactive dashboard creation.
- **SQL / Excel:** Used for data cleaning and handling alphanumeric IDs (like `seller_id`).
- **DAX (Data Analysis Expressions):** Created custom measures for total amounts and balances.
